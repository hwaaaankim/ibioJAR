# 일반 구매자 회원은 -> 일반회원가입을 진행 해 주시기 바랍니다.

# 관리자 승인 시 입력한 이메일과 문자를 통해 안내되니 정확한 정보를 입력 해 주시기 바랍니다.

# 만약 기존에 회원가입을 진행하신 분 이라면 -> 로그인을 진행 해 주세요.
